Cómo jugar a {{titulo}}
@switch (titulo) { @case ('Tabú') {
  • Objetivo: Hacer que tu equipo adivine una palabra sin usar palabras prohibidas.

  • Juego:

    • Un jugador (presentador) describe la palabra en la tarjeta sin usar las palabras prohibidas.
    • Su equipo intenta adivinar la palabra.
    • Si el presentador usa una palabra prohibida, la ronda termina o el equipo pierde un punto.
} @case ('Mímica y sonido') {
  • Objetivo: Hacer que tu equipo adivine una palabra o frase sin usar palabras.

  • Juego:

    • Mímica: Actúa la palabra o frase usando solo gestos.
    • Sonido: Emite sonidos para representar la palabra o frase, sin hablar.
    • Su equipo intenta adivinar la palabra o frase.
} @case ('Pintar') {
  • Objetivo: Hacer que tu equipo adivine una palabra o frase a través de dibujos.

  • Juego:

    • Un jugador dibuja la palabra o frase en la tarjeta sin usar letras ni números.
    • Su equipo intenta adivinar la palabra o frase basándose en el dibujo.
} @case ('Marcas') {
  • Objetivo: Hacer que tu equipo adivine la mayor cantidad de marcas o productos basado en una categoría.

  • Juego:

    • Un jugador nombra marcas o productos relacionados con una categoría dada, por ejemplo "Coches".
    • Su equipo intenta adivinar las marcas mencionadas, como "Volvo" o "Ferrari".
} @case ('Preguntas') {
  • Objetivo: Se tiene que dar una respuesta correcta a la pregunta formulada por el contrario.

  • Juego:

    • El equipo deberá tener un portavoz, que es el que se encargará de dar la respuesta final a la pregunta.
    • El equipo podrá debatir, pero el único que tiene voz para dar la respuesta es el portavoz.
} @case ('@currencias') {
  • Objetivo: Se debe acertar el mayor número de respuestas correctas para ir sumando puntos.

  • Juego:

    • El equipo contrario tomará la carta para comprobar las respuestas correctas.
    • El equipo deberá acertar todas las ocurrencias relacionadas con un tema en concreto. Por ejemplo: Francia: Torre Eiffel, Baguette, etc.
    • Se mostrará un número aleatorio, y si el número de respuesta coincide con ese número, esa respuesta vale doble.
} @default {

Algo va mal. Por favor, notifícamelo.

} }
